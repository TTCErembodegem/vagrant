Vagrant
=======

- Install Vagrant
- Clone this repo and..
- `vagrant up`

**ATTN: Hier geen migraties meer toevoegen**.  
Gebruik `Add-Migration` in `TTC-React-Back` voor alle nieuwe database wijzigingen.
